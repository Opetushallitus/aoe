# AOE Data Services

## OAI-PMH Provider

Integration service for metadata harvesting from external systems. Service interface implements the [OAI-PMH protocol](https://www.openarchives.org/OAI/2.0/openarchivesprotocol.htm).

- Java version: OpenJDK 17
- Running on port: 8001

## Build

```bash
mvn clean package
```

## Run

```bash
./mvnw spring-boot:run
```
